# Olá, eu sou o Abner Castro 👋

Estudante de **Engenharia da Computação pela PUC-Rio**, focado em integrar hardware e software para criar soluções automatizadas e sistemas inteligentes.

### 🚀 No que estou focado:
- **Automação de Infraestrutura:** Otimização de servidores de jogos e gestão de painéis Linux.
- **Sistemas Especialistas:** Desenvolvimento de lógica de compatibilidade de hardware usando Web Semântica.
- **IA & Diagnóstico:** Criando agentes inteligentes para resolução de conflitos de software.

### 🛠️ Projetos em Destaque:
* **Agente de Teste e Correção de Modpacks:** Automação via API para diagnóstico de falhas em servidores de jogos. Utiliza algoritmos de **pesquisa binária** e **IA** para identificar mods problemáticos de forma eficiente.
* **ProPC:** Plataforma desenvolvida em Django que utiliza ontologias (**RDF/SPARQL**) para garantir a compatibilidade técnica rigorosa entre componentes de hardware.
